ARMv8 Single Cycle Processor

This single cycle processor was written in Verilog as part of my Computer Architecture class in Spring 2019. The processor is capable of executing the following sets of instructions:

  • ADD (both Register and-Immediate type)
  • SUBtract (both Register and Immediate-type)
  • (bitwise) AND (R-type only)
  • (bitwise) ORR (R-type only)
  • CBZ (Branch if Zero)
  • B (Unconditional Branch)
  • MOVZ (Move with shift)
  • LDUR (Load into register)
  • STUR (Store into memory)

The ALU, Sign Extender, Control Block, Register File and Next PC modules were all designed separately and then brought together into a master module.
